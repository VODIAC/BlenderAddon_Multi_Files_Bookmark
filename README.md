# 🪐 Multi Files Bookmark

A lightweight Blender add-on for switching between multiple `.blend` project files in one Blender session, like using tabs in a browser.

Keep your active projects close, jump between files faster, and manage Blender work sessions with a clean dock-style bookmark bar.

## 🧭 What it does

**Multi Files Bookmark** adds a browser-like file switcher to Blender. You can save project files as bookmarks, view them as a compact dock bar, and switch between different `.blend` files with less file-browser detouring.

---

## ⚡ Key Features

### Browser-style project switching

Open and switch Blender project files through a tab-like bookmark workflow.

### Dock bar interface

Use a clean on-screen bar to keep your frequently used files visible and easy to access.

### Thumbnail-based bookmarks

Saved `.blend` files and their thumbnails are stored for quicker visual recognition.

### Toggle Bar shortcut

Show or hide the dock bar when you need a cleaner workspace.

### Customizable appearance

The latest release adds color customization, dark mode support, and a scaling range of **0.1–1.5**.

### Chinese / English support

Includes Chinese-English translation for a smoother experience in both language modes.

---

## 🌱 Why use it?

When working across several Blender files, opening projects from disk over and over can slow down the rhythm. Multi Files Bookmark keeps your project constellation in one place, so switching files feels closer to changing browser tabs than digging through folders.

## 🛠️ Installation

### From Blender Extensions

1. Open Blender.
2. Go to **Edit > Preferences > Get Extensions**.
3. Search for **Multi Files Bookmark**.
4. Install and enable the add-on.

### From GitHub Releases

1. Download the latest release package.
2. In Blender, open **Edit > Preferences > Add-ons**.
3. Click **Install from Disk**.
4. Select the downloaded `.zip` file.
5. Enable **Multi Files Bookmark**.

[Download from GitHub Releases](https://github.com/VODIAC/BlenderAddon_Multi_Files_Bookmark/releases)

---

## 🌀 Typical Workflow

1. Add frequently used `.blend` files to the bookmark list.
2. Use the dock bar to switch between project files.
3. Adjust the bar color, scale, or dark mode to match your workspace.
4. Hide the bar with the shortcut when you want a cleaner view.
