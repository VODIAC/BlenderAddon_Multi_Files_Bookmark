## ✨ Features

- **Bookmark multiple `.blend` files** and keep them ready in one panel.
- **Switch projects quickly** without digging through folders again and again.
- **Chrome-like tab workflow** for a cleaner multi-file experience.
- **Customizable panel style**, including dark mode and custom colors.
- **Adjustable UI scale**, updated to a practical `0.1–1.5` range.
- **Chinese and English translation** for a smoother bilingual workflow.

## 🧭 Workflow

1. Add your frequently used Blender files as bookmarks.
2. Open the bookmark panel inside Blender.
3. Click a saved file to switch/open it quickly.
4. Adjust the panel color, scale, and shortcut behavior as needed.

## 📦 Installation

### Install from Blender Extensions

Download or install it from the official Blender Extensions page:

https://extensions.blender.org/add-ons/multi-files-bookmark/

### Install from GitHub Release

1. Open the latest release page:  
   https://github.com/VODIAC/BlenderAddon_Multi_Files_Bookmark/releases
2. Download the add-on `.zip` file.
3. In Blender, go to `Edit > Preferences > Add-ons`.
4. Click `Install from Disk`.
5. Select the downloaded `.zip` file and enable the add-on.
