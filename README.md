## 🧭 What it does

**Multi Files Bookmark** helps you manage Blender projects like browser tabs. Add frequently used `.blend` files to a dock bar, view them with thumbnails, and jump between projects without repeatedly digging through folders.

---

## ⚡ Key Features

- **Tab-style project switching**: open saved `.blend` files from a compact dock bar.
- **Project bookmarks**: keep frequently used files ready for quick access.
- **Thumbnail preview**: recognize files visually instead of only reading names.
- **Customizable UI**: adjust scale, color style, and dark mode behavior.
- **Toggle shortcut**: show or hide the dock bar when you need more viewport space.
- **Chinese / English support**: cleaner experience in both language modes.

## 🛠️ Installation

### From Blender Extensions

1. Open Blender.
2. Go to **Edit > Preferences > Get Extensions**.
3. Search for **Multi Files Bookmark**.
4. Install and enable the add-on.

### Manual installation

1. Download the latest package from [Releases](https://github.com/VODIAC/BlenderAddon_Multi_Files_Bookmark/releases).
2. Open **Edit > Preferences > Add-ons**.
3. Click **Install from Disk**.
4. Select the downloaded `.zip` file and enable the add-on.

---

## 🌀 Basic Workflow

1. Enable **Multi Files Bookmark** in Blender.
2. Add your frequently used `.blend` projects to the dock.
3. Use the tab-style bar to switch files quickly.
4. Customize the dock appearance and shortcut behavior in the add-on settings.
